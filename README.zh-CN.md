# MCU Showcase

[English](README.md) | [中文](README.zh-CN.md)

微控制器（MCU）开发示例项目合集。
本仓库按四个类别组织：

1. **app** — 应用层示例，使用驱动或 RTOS 功能。
2. **rtos** — RTOS 示例，包括任务调度、队列、信号量和定时器。
3. **bootloader** — Bootloader 和固件升级示例，包括串口/USB/OTA 升级演示。
4. **driver** — 硬件驱动，如 GPIO、UART、SPI、I2C、ADC/DAC。

> 注意：所有项目仅供学习、参考和快速原型验证之用。
